# ejemploStackLinkedLists

El presente proyecto está creado en java.

Se creó un "ejemplo" de navegador web, donde se utiliza la estructura de datos conocida
como "Stack" para representar los botones "adelante" y "atrás". 

Esta estructura se relacionó a la vez un una lista ligada, para que no hubiese la necesidad
de crear un array con x número de elementos, si no poder hacer que crezca n veces.
